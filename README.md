# Technical Support Ticket System

Desktop application built in **C# (Windows Forms)** with **MySQL** for managing technical support tickets efficiently.

---

## Features

- Create and manage support tickets  
- View and filter tickets by status and priority  
- Update ticket status in real time  
- Delete multiple tickets  
- Technician management (CRUD)  
- Interactive dashboard with ticket statistics  

---

## Technologies

- C#
- .NET (Windows Forms)
- MySQL
- ADO.NET
